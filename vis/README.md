# Paper #3878 Supplementary Materials

## *Why (Not) ReacTIVision: Charting Emerging Challenges and Opportunities for Building Tangible User Interfaces with Computer Vision

This supplementary website is an interactive visualization we developed to explore the emerging trends and schemes found in the literature. Please open the following files in a web browser to view. Please see the supplementary video for a demonstration.

- `visualization.html`: Interactive visualization that lets readers filter and sort based on the 17 dimensions we used to code the papers. Click on a paper label to view its details.

- `table.html`: Complete corpus table showing the codes assigned to each paper to lend a complete picture. Readers can sort the list of papers based on any of the codes. Click on a row to view the paper details.

- `Paper Coding.csv`: View the code table in a spreadsheet format.
